EIA Country Energy Search

Search one country-level EIA international energy series, graph the selected result, and download observations as CSV.

Type a country and energy topic. Examples: Brazil energy consumption · Jordan electricity generation · Mexico natural gas production · Japan oil consumption

About this data

This page calls a Vercel backend route, which calls the U.S. Energy Information Administration API using the private EIA_API_KEY stored in Vercel. The API key is not placed in this public HTML file.

Coverage shown here is calculated from actual observations returned for the selected EIA series. That prevents the old mismatch where a variable list could show a shorter range than the full series.

This version is intentionally single-country only. It does not include multi-country comparison mode.
