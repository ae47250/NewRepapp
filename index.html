EIA Energy Data Search

Search country-level EIA international energy data, compare countries, graph the selected series, and download the data as CSV.

Examples: Brazil energy consumption · compare Canada and Mexico natural gas production · United States electricity generation

About this data

This page calls a Vercel backend route, which calls the U.S. Energy Information Administration API using the private EIA_API_KEY stored in Vercel. The API key is never placed in this public HTML file.

Coverage shown in the results is calculated from actual returned observations for each displayed series. That prevents the old problem where a variable list showed a shorter range than the selected full series.
